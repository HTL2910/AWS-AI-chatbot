# 🔨 Hướng Dẫn Build Tool

Hướng dẫn chi tiết cách build và phát triển SafeGraph AI VS Code Extension.

---

## 📋 Yêu Cầu Hệ Thống

### Chung
- **Git:** 2.0 trở lên
- **Node.js:** 18.x trở lên (bắt buộc cho VS Code extension)
- **Python:** 3.8 trở lên (cho Streamlit app)

### Windows
- PowerShell 5.0+ hoặc Command Prompt
- Visual C++ Build Tools (nếu cần compile native modules)

### macOS / Linux
- GCC hoặc Clang compiler
- Make utility

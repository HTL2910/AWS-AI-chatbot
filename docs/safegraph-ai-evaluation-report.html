Điểm mạnh kỹ thuật

  • Extension compile sạch với TypeScript và có VSIX safegraph-ai-0.17.0.vsix.
  • Chat provider có task state, status bar, local workspace tools, command feedback và context handling.
  • Local file tools có kiểm tra path escape workspace, giúp giảm rủi ro đọc file ngoài project.
  • Inline completion có cấu hình enable/disable, manual/automatic trigger và debounce.
  • Prompt hệ thống trong agent chat có nhiều quy tắc thực dụng cho coding workflow, frontend, backend và verification.

Rủi ro và khoảng trống

  1. Command policy quá rộng. Trong mode safe, shell chaining/redirection không tự động bị hỏi lại nếu không match danh sách regex nguy hiểm.
  2. Contribution points chưa khớp runtime. package.json khai báo openAgentManager, openArtifactViewer, openAuditLog nhưng activation code chưa register đầy đủ.
  3. README overclaim. Tài liệu mô tả multi-agent/artifact/background execution như tính năng hoàn chỉnh, trong khi code hiện giống scaffold hoặc tích hợp một phần.
  4. Test coverage thấp. Build pass chưa đủ để tin vào behavior runtime của VS Code extension, diff apply, Bedrock error handling và command runner.
  5. Artifact parsing dễ lỗi. Generator parse trực tiếp JSON.parse(response.text), dễ fail khi model trả markdown hoặc text phụ.

================================================================================
🔑 LẤY AWS CREDENTIALS
================================================================================

Cách 1: Sử Dụng Setup Script (Dễ Nhất)
$ python setup_credentials.py

Cách 2: Cấu Hình Thủ Công
1. Vào https://console.aws.amazon.com
2. Tìm IAM → Users → Chọn user
3. Security credentials → Create access key
4. Copy Access Key ID và Secret Access Key
5. Tạo file .env:
   AWS_ACCESS_KEY_ID=your_key
   AWS_SECRET_ACCESS_KEY=your_secret
   AWS_DEFAULT_REGION=us-east-1

================================================================================
🎯 CẤU HÌNH QUYỀN BEDROCK
================================================================================

1. Vào AWS IAM Console
2. Chọn user của bạn
3. Add permissions → Attach policies directly
4. Tìm "AmazonBedrockFullAccess"
5. Chọn → Add permissions

================================================================================
⚠️ LỖI THƯỜNG GẶP
================================================================================

Lỗi: ModuleNotFoundError
Giải pháp: pip install -r requirements.txt

Lỗi: InvalidSignatureException
Giải pháp: Kiểm tra API Key trong .env

Lỗi: AccessDenied
Giải pháp: Thêm AmazonBedrockFullAccess policy trong IAM

Lỗi: ModelNotFound
Giải pháp: Thử region khác (us-west-2 hoặc eu-west-1)

🛡️ BẢO MẬT
================================================================================

✅ NÊN LÀM:
- Lưu credentials trong .env
- Sử dụng IAM User (không Root)
- Giới hạn quyền (chỉ Bedrock)
- Rotate keys định kỳ
- Sử dụng MFA

❌ KHÔNG NÊN LÀM:
- Không commit .env lên Git
- Không chia sẻ Secret Key
- Không hardcode credentials
- Không sử dụng Root Account
- Không để credentials trong logs

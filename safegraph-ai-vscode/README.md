# Safegraph AI (MVP)

This folder is a minimal VS Code extension scaffold for the MVP plan.

## Dev

1. `npm i`
2. `npm run build`
3. Open this folder in VS Code and press `F5` (Run Extension) to launch an Extension Development Host.

Week 1 behavior: sidebar chat view echoes user messages back from the extension host.

## Dock to Right (like Cursor)

VS Code does not let extensions force a view to always appear on the right automatically.
You can move it once and VS Code will remember the layout:

- Click `Dock Right` in the chat header, then pick `Secondary Side Bar` in the Move View UI.
- Or right-click the Safegraph AI icon in the Activity Bar and choose `Move View to Secondary Side Bar`.

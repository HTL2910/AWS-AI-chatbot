# ⚡ Quick Start

Bắt đầu trong 5 phút!

## 1️⃣ Cài Đặt

```bash
pip install -r config/requirements.txt
```

## 2️⃣ Cấu Hình

```bash
python src/setup_credentials.py
```

## 3️⃣ Chạy

```bash
streamlit run src/app.py
```

## 4️⃣ Chat!

- Mở Sidebar (bên trái)
- Click "🔧 Cấu Hình Kết Nối"
- Bắt đầu chat!
